# 福井県オープンデータまとめ

福井県の漁業や地魚に関するオープンデータを収集・公開しているプロジェクトです。地魚の写真や情報を閲覧できます。

## デモ
[福井の地魚一覧](https://code4fukui.github.io/fukui-pref/fukuijizakana/)

## 機能
- 福井県の地魚の写真を一覧表示
- 地魚の詳細情報（魚名、魚種、写真URL）を提供

## データ・API
このプロジェクトのデータは、[福井県オープンデータライブラリ](https://www.pref.fukui.lg.jp/doc/toukei-jouhou/opendata/)から取得しています。
