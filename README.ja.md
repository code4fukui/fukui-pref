# 福井県オープンデータまとめ

福井県のオープンデータを活用したプロジェクトです。地魚の情報や写真を公開しています。

## デモ
[福井の地魚一覧](https://code4fukui.github.io/fukui-pref/fukuijizakana/)

## 機能
- 地魚の写真一覧の表示
- 地魚の詳細情報（魚名、魚種、写真URL）の提供

## データ・API
本プロジェクトで使用しているデータは、[福井県オープンデータライブラリ](https://www.pref.fukui.lg.jp/doc/toukei-jouhou/opendata/)から取得したものです。
